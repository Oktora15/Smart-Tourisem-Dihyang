
# Smart Tourism Website Design

A Vite + React project for a Smart Tourism web app. The original design is available in Figma: https://www.figma.com/design/L2aCRgHjN7MdhgnO9ckuKl/Smart-Tourism-Website-Design

## Development

This project uses `pnpm` (recommended) but will work with `npm`/`yarn`.

Install dependencies and start the dev server:

```bash
pnpm install
pnpm dev
```

Or with npm:

```bash
npm install
npm run dev
```

## Build

```bash
pnpm build
pnpm preview
```
